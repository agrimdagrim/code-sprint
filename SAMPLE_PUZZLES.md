# Sample Crossword Puzzles for Exolve Format

Copy and paste these puzzles into your Firebase Firestore document at:
`competitions/sprint_001` field `puzzleSet` (as an array)

Each puzzle is a separate string in the array. Test 1-2 puzzles first before uploading all.

# HOW TO ADD THESE TO FIREBASE

1. Go to Firebase Console: https://console.firebase.google.com/
2. Select your project: `crossword-sprint`
3. Go to **Firestore Database**
4. Navigate to: `competitions` â†’ `sprint_001`
5. Click **Edit** (pencil icon) on that document
6. Add a new field:
   - Field name: `puzzleSet`
   - Type: `Array`
7. Add each puzzle as a string element in the array
   - Click **Add array item**
   - Paste the entire puzzle text (everything between `exolve-begin` and `exolve-end`)
   - Repeat for all 10 puzzles

**Important:** Each puzzle should be a separate string element in the array, with the exact Exolve format (no extra spaces at the beginning or end).
